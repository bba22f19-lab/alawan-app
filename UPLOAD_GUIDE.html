Al Awan Enterprises — Upload Guide

idreesawan1812 ke liye — GitHub Pages Setup

✅ Aapki App ka Final Link (upload ke baad):

https://idreesawan1812.github.io/alawan-app
1

GitHub par Login karein

Browser mein github.com kholen aur idreesawan1812 account se login karein.

✅ Agar account nahi bana toh pehle banao — bilkul free hai
2

Naya Repository banayein

  1. Top right mein "+" icon → "New repository"
  2. Repository name mein likhen:
alawan-app
  1. "Public" select karein
  2. "Create repository" click karein
⚠️ Name exactly alawan-app likhein — chota case, dash ke saath
3

ZIP extract karein aur files upload karein

AlAwan-App-Files.zip extract karein — ye files mileingi:

  • index.html — main dashboard
  • manifest.json — app settings
  • sw.js — offline support
  • icon.svg — app logo
  • icon-192.png — mobile icon
  • icon-512.png — large icon
  1. Repository mein "uploading an existing file" link click karein
  2. Saari 6 files select karke drag karein ya choose karein
  3. Neeche "Commit changes" click karein
❌ ZIP file upload mat karein — andhar ki 6 files alag upload karein
4

GitHub Pages Enable karein

  1. Repository mein "Settings" tab click karein
  2. Left sidebar mein "Pages" click karein
  3. Source: "Deploy from a branch" select karein
  4. Branch: "main" → "/ (root)" select karein
  5. "Save" click karein
⏳ 2-3 minute wait karein phir link ready ho jayega
5

App Install karein — Mobile aur PC

Ye link kisi bhi device par kholen:

https://idreesawan1812.github.io/alawan-app
🤖

Android

Chrome → 3 dots menu → "Add to Home Screen" → Install

🍎

iPhone

Safari → Share (□↑) → "Add to Home Screen" → Add

💻

PC

Chrome → Address bar ka install icon (⊕) → Install
